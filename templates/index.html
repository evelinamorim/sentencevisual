Sentence Annotation Visualizer

Instructions

Each time expression is shown inside a yellow box, and their order is the same as they appear in the text. The first is shown first, etc.

Also, time expression has the color (yellow), except for Duration and Set type, in which case it presents the color (orange).

By hovering over an event, it is possible to see its attributes.

Events with a darker dashed border are of type Intensional Action or Intensional State, i.e., they are events that may or may not happen.

Please select a file and click on the time expression or event to display its sentence and respective attributes.
